# Color Transfer V1

完全在本地运行的强参考配色迁移工具。上传原图和参考图后，程序会迭代匹配参考图的完整三维颜色分布，同时保留原图构图、分辨率与细节位置。

## Windows 启动

双击 `start.bat`。首次启动会创建 `.venv` 并安装依赖，之后可以断网启动。浏览器默认打开 `http://127.0.0.1:7860`，关闭启动窗口即可停止服务。

需要 Python 3.11，支持 JPG、JPEG、PNG、WEBP。带透明通道的 PNG 会保留 Alpha，所有图片处理均在本机完成，不会上传互联网。

## 参数

- 色彩迁移强度：0 保留原图，1 使用完整迁移结果，默认 1.0。
- 亮度保护：数值越高越接近原图亮度；默认 0.0，完整采用参考图明暗配色。
- 饱和度：0 为灰度，1 不调整，2 增强饱和度，默认 1.0。

## 开发测试

```powershell
.\.venv\Scripts\python.exe -m pytest
.\.venv\Scripts\python.exe app.py
```

V1.1 使用 Pitié–Kokaram–Dahyot 思路的迭代式三维颜色概率分布迁移，并通过确定性 3D LUT 分块应用到原图全尺寸。它会强烈替换整体配色，但仍不理解人物、天空等语义；选择极端参考图时结果也会相应极端。权威需求与 TODO 见根目录的 `Color_Transfer_V1_开发文档与TODO.md`。
